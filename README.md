# clone_instagram
 projeto em NodeJs e MongoDB clonando o instagram\

 ## Para executar o projeto segue os passos a baixo:
 ```
> cd backend\
> npm i\
> yarn dev\
 ```
 
 ## depois abrir outro terminal:
 ```
> cd frontend\
> npm i\
> yarn start\
 ```
 
 ### para acessar o projeto: http://localhost:8080
